## 仿知乎页面多人博客平台


## 前端：React+redux+react-router-dom+ES6+Antd
## 后端：Koa+Mongoose
### 运行方法
#### 1.获取仓库
```javascript
git clone https://github.com/NovemberChopin/react-blog.git
```

#### 2.安装客户端依赖
```javascript
cd react-blog & npm install 
```

#### 3.安装服务端依赖
```javascript
cd server & npm install 
```

#### 4.打包前端代码
```javascript
npm run build
```

#### 5.打开mongoDB数据库后  启动服务端
```javascript
npm run dev
```
