## 慕课网react前端课程视频代码

### 技术栈
```javascript
react + redux + react-router-dom + antd + express
```
### 演示地址 [仿简书博客](http://39.96.175.74:3000/)

### 运行方法
#### 1.获取仓库
```javascript
git clone https://github.com/NovemberChopin/react-blog-jianshu.git
```

#### 2.安装依赖
```javascript
npm install
```
#### 3.运行
```javascript
npm run start
```
